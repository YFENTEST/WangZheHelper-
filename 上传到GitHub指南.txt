========================================================
  上传到 GitHub 并自动编译 - 超简单步骤
========================================================

【第 1 步】在你的新仓库页面

1. 点击 "uploading an existing file"（上传现有文件）
   或者点击 "Add file" → "Upload files"

2. 把这些文件和文件夹**全部拖进去**：
   - .github 文件夹（重要！）
   - Tweak.x
   - Makefile
   - control
   - WangZheHelper.plist
   - wzhelperprefs 文件夹
   - README.md
   - LICENSE
   - .gitignore

3. 滚动到底部，点 "Commit changes"（提交更改）

【第 2 步】查看编译进度

1. 上传完成后，点击仓库顶部的 "Actions" 标签

2. 你会看到一个正在运行的任务（黄色圆圈转圈）

3. 等待 5-10 分钟，直到变成绿色✓（编译成功）

【第 3 步】下载 deb 文件

1. 点击那个绿色✓的任务

2. 滚动到底部，找到 "Artifacts"（产物）

3. 点击 "WangZheHelper-deb" 下载

4. 解压下载的 zip 文件，里面就是 .deb 文件！

========================================================

【以后修改代码】

每次修改代码后：
1. 在 GitHub 网页上直接编辑文件
2. 或者上传新文件覆盖
3. 提交后自动编译
4. 从 Actions 下载新的 deb

========================================================

【手动触发编译】

如果没有修改代码，只是想重新编译：
1. 点击 "Actions" 标签
2. 左侧选择 "Build iOS Tweak"
3. 右侧点击 "Run workflow" → "Run workflow"
4. 等待编译完成，下载 deb

========================================================

就这么简单！不需要安装任何东西！
